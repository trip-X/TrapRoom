# TrapRoom

一款基于 Cocos Creator 开发的策略闯关游戏。玩家需要在布满陷阱的房间中，通过移动和转向来躲避地刺、爆炸、敌人等障碍，找到出口通关。

## 游戏特性

- **21 个精心设计的关卡** — 从简单到复杂，逐步提升难度
- **丰富的实体系统** — 玩家、木骷髅、铁骷髅、门、地刺、爆炸桶、烟雾等
- **状态机驱动** — 每个实体拥有独立的状态机，支持空闲、攻击、转向、格挡、死亡等多种状态
- **回合制移动** — 玩家每操作一步，敌人同步行动
- **撤销/重做** — 支持操作回退，降低试错成本
- **屏幕震动与烟雾特效** — 增强战斗打击感与视觉反馈

## 技术栈

- **引擎**：Cocos Creator 3.8.7
- **语言**：TypeScript
- **架构模式**：状态机（FSM）+ 事件驱动（Event-Driven）+ 单例管理器

## 项目结构

```
assets/
├── Base/                 # 基础类：实体管理器、状态机、单例等
├── Enums/                # 全局枚举定义
├── Levels/               # 21 个关卡数据配置
├── Runtime/              # 运行时管理器：数据、事件、淡入淡出、资源
├── Scripts/
│   ├── Burst/            # 爆炸桶逻辑与状态机
│   ├── Door/             # 门逻辑与状态机
│   ├── IronSkeletion/    # 铁骷髅敌人逻辑与状态机
│   ├── Player/           # 玩家逻辑与状态机
│   ├── Scene/            # 场景管理：战斗、加载、开始
│   ├── Smoke/            # 烟雾特效逻辑
│   ├── Spikes/           # 地刺陷阱逻辑与状态机
│   ├── Tile/             # 瓦片地图管理
│   ├── UI/               # UI 控制：方向键、菜单、震动
│   ├── Utils/            # 工具函数
│   └── WoodenSkeleton/   # 木骷髅敌人逻辑与状态机
├── Sences/               # 游戏场景：Battle、Loading、Start
├── animations/           # 动画资源
└── resources/            # 纹理图集等资源
```

## 运行项目

1. 安装 [Cocos Dashboard](https://www.cocos.com/creator-download)
2. 在 Cocos Dashboard 中添加 Cocos Creator 3.8.7 编辑器版本
3. 通过 Dashboard 导入本项目
4. 打开项目后，选择 `Start` 场景预览运行

## 操作方式

| 操作 | 功能 |
|------|------|
| ↑ ↓ ← → | 上下左右移动 |
| 左转 | 角色向左转向 |
| 右转 | 角色向右转向 |
| 撤销 | 回退上一步操作 |
